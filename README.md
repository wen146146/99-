# 99乘法表

Python实现的99乘法表，使用嵌套循环和格式化输出实现。

## 功能特点
- 输出完整的99乘法表
- 格式清晰，自动对齐
- 代码简洁，易于理解

## 运行方法
1. 确保已安装Python 3
2. 克隆仓库：`git clone https://github.com/wen146146/99-.git`
3. 进入目录：`cd 99-`
4. 运行脚本：`python multiplication_table.py`

## 代码示例
```python
# 99乘法表
for i in range(1, 10):
    for j in range(1, i + 1):
        # 格式化输出，保持对齐
        print(f"{j} × {i} = {i*j:2}", end="  ")
    print()  # 换行
```
